# 🌦️ Weather Forecast App

## 📌 Description

The Weather Forecast App is a dynamic web application that provides real-time weather updates for any location using API integration. Users can search for a city and instantly view accurate weather details such as temperature, humidity, wind speed, and overall conditions. The project demonstrates practical implementation of web technologies and API usage to build a real-world application.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💨 Wind speed and humidity details
* ☁️ Weather condition overview (Clear, Rain, Clouds, etc.)
* 📱 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Weather API (e.g., OpenWeatherMap)

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/weather-forecast.git

2. Open the project folder:
   cd weather-forecast

3. Open `index.html` in your browser
